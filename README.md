"# Digital-Projector" 

Digital Projector is a lightweight, cross-platform web experience that lets you project video content onto any wall. Load a local video or the included sample clip, then use fullscreen mode to fill the display.

## Features

- Local video selection (no uploads required)
- Sample video playback for quick setup
- Fullscreen wall projection mode
- Keyboard controls for play/pause, mute, and fullscreen

## Quick start

1. Start a local server:
   ```bash
   python3 -m http.server 8000
   ```
2. Open the experience in your browser:
   ```
   http://localhost:8000
   ```

## Controls

- **Select video**: choose a local file to play
- **Play/Pause**: toggle playback
- **Mute/Unmute**: control audio
- **Fullscreen**: maximize for projection
- **Keyboard shortcuts**: `Space` (play/pause), `M` (mute), `F` (fullscreen)

## Install as an app

Digital Projector is a Progressive Web App (PWA), so you can install it on Android, iOS, and desktop browsers.

### Android (Chrome/Edge)

1. Open the site in Chrome.
2. Tap the **Install app** prompt or use the menu → **Install app**.

### iOS (Safari)

1. Open the site in Safari.
2. Tap **Share** → **Add to Home Screen**.

### Desktop (Chrome/Edge/Safari)

1. Open the site in a supported browser.
2. Click the install icon in the address bar or use the browser menu → **Install Digital Projector**.
